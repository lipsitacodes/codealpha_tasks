## рЯНФ Project вАФ HappyBites

> *"Good Food. Happy You."*

**HappyBites** is a modern Indian food delivery mobile app designed for urban professionals. Built with a warm orange and cream white design system, it offers a seamless food ordering experience similar to Zomato and Swiggy вАФ but cleaner, faster, and more transparent.

### рЯОѓ Problem Solved
Users struggle with hidden fees, cluttered interfaces and poor order tracking on existing food delivery apps. HappyBites solves all three with transparent pricing, minimal UI and live order tracking.
